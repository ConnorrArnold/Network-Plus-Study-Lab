## 📚 Key Topics

**Networking Concepts**
- OSI and TCP/IP models
- IP addressing, subnetting, and CIDR notation
- Routing protocols (OSPF, BGP, RIP)
- Switching concepts (VLANs, STP, trunking)
- DNS, DHCP, NAT, and common ports/protocols

**Network Implementation**
- Wireless standards (802.11 a/b/g/n/ac/ax)
- Cable types and physical infrastructure
- Network device roles (routers, switches, firewalls, APs)
- Virtual networking (VLANs, VPNs, SD-WAN)

**Network Security**
- Common attack types (DoS, MITM, spoofing, phishing)
- Firewall rules and ACLs
- Network hardening and segmentation
- Authentication protocols (RADIUS, TACACS+)

**Network Troubleshooting**
- Troubleshooting methodology
- Using tools: ping, traceroute, nslookup, netstat, Wireshark
- Common connectivity issues and resolutions
